 Skeptical News Analyzer

An AI-powered news analysis tool that detects bias, tone, and claims in online news articles.  
Built with Streamlit, Python, and Groq AI.

---

 Features
- Paste any article URL and get instant AI-based analysis.
- Bias Detection ‚Üí Detect potential bias in the writing.
- **Tone Analysis ‚Üí Identify tone (neutral, sensational, positive, negative).
- Claim Extraction ‚Üí Highlight factual claims made in the article.
- Interactive Dashboard ‚Üí View results in a clean, card-style interface.
- Delete Stored URLs ‚Üí Manage and remove analyzed articles.
- Fast Processing‚Üí Powered by **Groq AI** for real-time analysis.
